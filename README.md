# 자소서 글자수 계산기 (Jasoseo Byte Counter)

> 자기소개서 작성을 위한 실시간 글자수 및 바이트 계산 크롬 확장 프로그램

[![Version](https://img.shields.io/badge/version-1.3-blue.svg)](https://github.com/sa03134/jasoseo-counter)
[![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-green.svg)](https://chrome.google.com/webstore)

## 📋 주요 기능

### ✨ 실시간 계산
- **공백포함 글자수**: 실시간 글자 수 계산
- **바이트 계산**: EUC-KR 방식의 정확한 바이트 계산 (잡코리아, 사람인 등과 동일)
- **즉시 확인**: 입력과 동시에 자동 계산

### 📄 문항 관리
- **기본 4개 문항**: 일반적인 자소서 문항 수
- **문항 추가**: 필요시 무제한 문항 추가 (5번부터 삭제 가능)
- **자동 저장**: 브라우저를 껐다 켜도 내용 유지

### 📋 복사 기능
- **📄 텍스트 복사**: 각 문항의 전체 텍스트 복사
- **字 글자수 복사**: 공백포함 글자수만 복사
- **B 바이트 복사**: 바이트 수만 복사
- **전체 복사**: 모든 문항의 정보를 한번에 복사

### 🗑️ 관리 기능
- **내용 지우기**: 각 문항의 내용만 초기화
- **문항 삭제**: 추가된 문항(5번~) 삭제
- **전체 삭제**: 모든 문항 내용 일괄 삭제

## 🚀 설치 방법

### Chrome 웹 스토어에서 설치 (추천)
*곧 출시 예정*

### 수동 설치
1. 이 저장소를 다운로드하거나 클론합니다
   ```bash
   git clone https://github.com/sa03134/jasoseo-counter.git
   ```

2. Chrome 브라우저에서 `chrome://extensions/` 접속

3. 우측 상단의 "개발자 모드" 활성화

4. "압축해제된 확장 프로그램을 로드합니다" 클릭

5. 다운로드한 폴더 선택

## 💡 사용 방법

1. **확장 프로그램 아이콘 클릭**
   - Chrome 우측 상단의 확장 프로그램 아이콘 클릭

2. **자소서 작성**
   - 각 문항에 자소서 내용 입력
   - 실시간으로 글자수/바이트 확인

3. **복사 및 활용**
   - 필요한 정보를 버튼 클릭으로 간편하게 복사
   - 기업 채용 사이트에 붙여넣기

## 🔧 기술 스택

- **Vanilla JavaScript**: 순수 자바스크립트로 구현
- **Chrome Extension Manifest V3**: 최신 확장 프로그램 규격
- **LocalStorage**: 자동 저장 기능
- **CSS3**: 모던하고 깔끔한 UI

## 📊 바이트 계산 방식

EUC-KR 인코딩 기준으로 계산합니다 (잡코리아, 사람인 등 주요 채용 사이트와 동일):
- **영문/숫자/기본 기호**: 1바이트
- **한글/특수문자**: 2바이트
